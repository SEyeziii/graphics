# graphics

Выполнение лабораторных работ по предмету "компьютерная графика"

1 лабораторная работа включает в себя 3 задачи, которые представлены в файлах 1_1.py 1_2.py 1_3.py 

1_1
Поворот одного отрезка, координаты задаются с клавиатуры. Использовать стандартный алгоритм Брезенхема.
1_2
Поворот контура простой фигуры (треугольник, прямоугольник).
Интерактивный ввод точки центра поворота мышью.
1_3
Поворот сложного многоугольника (звезда, произвольная ломаная)
с возможностью анимированного вращения (автоматическое изменение угла по
таймеру). Реализовать сглаживание отрезков (антиалиасинг).

далее по ходу выполнения работ будут добавляться новые файлы с соответствующими заданиями и описаниями задач
